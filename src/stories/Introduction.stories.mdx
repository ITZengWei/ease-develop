# 欢迎来到我的 smalllb 组件库

这个 UI 组件库是 通过 React + Hooks + TypeScript + Storybook 联合打造的 
一起开始奇妙之旅吧

<div className="card">
  <h3 class="title">通过 npm 安装</h3>
